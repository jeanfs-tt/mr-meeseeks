## What is this?

This project demonstrates how **AGENTS.md** and **Skills** can standardize code generated by GitHub Copilot across different models.

The helper functions in `src/helpers/` are the "gold standard" — generated **with** AGENTS.md guidance. During the demo, we generate helpers **without** AGENTS.md, then **with** it, and compare the quality scores.
