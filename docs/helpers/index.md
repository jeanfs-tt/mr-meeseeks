# Helpers

A collection of pure, validated, well-documented utility functions.

Every helper in this library follows the **Mr. Meeseeks Standard**:

| Criterion | Requirement |
|-----------|-------------|
| JSDoc | `@description`, `@param`, `@returns`, `@example` (×2), `@throws`, `@since` |
| Input Validation | Throws `TypeError` with `[functionName] message` format |
| Purity | No side effects, no input mutation |
| Types | No `any`, explicit return types, `readonly` for arrays |
| Tests | Co-located `.test.ts` with happy path, edge cases, error cases |
| Naming | camelCase, verb-prefixed (`format`, `get`, `is`, `has`, `to`, `parse`) |
